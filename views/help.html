Mètodes de l'API

URL Mètode Funció
/login POST S'hi envia l'usuari i la contrasenya en el body JSON. Retorna el token JWT a fer servir
/logout GET Elimina la cookie JWT
/aula/list GET Llista les aules del sistema (necessita el token o donarà error)
/aula/{numero}/status GET Llista les màquines en marxa de l'aula 'número' (necessita el token o donarà error)
Ex. Aula 309 : /aula/309/status
/aula/{numero}/stop POST Encara no fa res... (necessita el token o donarà error)
